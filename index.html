Detail Proyek WebGIS Wisata Alam Yogyakarta

WebGIS ini dirancang untuk menampilkan persebaran destinasi wisata alam di Daerah Istimewa Yogyakarta secara informatif, menarik, dan mudah digunakan. Struktur tampilannya sudah cocok untuk tugas kuliah karena mencakup peta, filter, daftar lokasi, popup detail, dan informasi proyek.

Komponen Utama

  • Peta interaktif berbasis Leaflet dan OpenStreetMap.
  • Filter kategori, kabupaten, pencarian kata kunci, dan minimal rating.
  • Daftar lokasi interaktif yang terhubung langsung ke peta.
  • Layer batas wilayah sederhana sebagai elemen visual presentasi.
  • Rute garis lurus dari Titik Nol Yogyakarta ke lokasi terpilih.

Yang Bisa Kamu Edit

  • Ganti data di variabel wisataGeoJSON dengan data aslimu.
  • Ganti judul, deskripsi, nama kelompok, dan tujuan proyek.
  • Tambah polygon, foto, statistik, atau data dari file GeoJSON eksternal.
  • Kalau punya data QGIS, ekspor ke GeoJSON lalu tempel ke template ini.
